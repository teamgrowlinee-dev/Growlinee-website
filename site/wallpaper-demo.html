Animeeritud Taustapilt

Elegantne must liivadüünide tekstuur sädelevate osakestega. Sobib suurepäraselt veebilehe taustaks.

Tagasi avalehele

Hero Sektsiooniga Demo

See versioon on optimeeritud Hero sektsiooni jaoks - sisaldab gradiendikihi, mis sobib paremini olemasoleva disainiga.

Kuidas kasutada

1. Hero sektsiooni jaoks

Asenda Hero.tsx failis FluidHeroBackground komponendiga AnimatedHeroBackground:

// Hero.tsx
import { AnimatedHeroBackground } from "@/components/sections/AnimatedHeroBackground";

// Asenda:
// <FluidHeroBackground />
// Uuega:
<AnimatedHeroBackground />

2. Muu sektsiooni taustaks

Kasuta AnimatedWallpaper komponenti mis tahes sektsiooni ümber:

import AnimatedWallpaper from "@/components/ui/AnimatedWallpaper";

<AnimatedWallpaper className="min-h-screen">
  <div>Sinu sisu siia</div>
</AnimatedWallpaper>

3. Osakeste arvu muutmine

Sädelevate osakeste arvu saab muuta particleCount propiga:

<AnimatedWallpaper particleCount={100}>
  ...
</AnimatedWallpaper>
